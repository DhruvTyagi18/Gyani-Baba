## Setup
To get started, follow these steps:

1. Clone the repository
    ```sh
    git clone https://github.com/your-username/gyani-baba.git
    cd gyani-baba
    ```

2. Install the required dependencies
    ```sh
    npm install
    ```

3. Start the development server
    ```sh
    npm start
    ```
